#Scottish Climate Survey App

This app is designed to visualise and analyse the Scottish Climate Survey 2025 data

To run the app, open the file /src/SCS_app_current.R and press Run App in RStudio. 

Select the survey data xlsx file for analysis. 

The plots-raw tab computes the weighted proportions by primary and secondary grouping variables

The plots-grouped tab groups questions for which respondents could have selected one or more options for a question

The Statistics tab runs statistical tests with multiple comparisons corrections. Users can group one or more levels for both the Variable of Interest (question) and the Primary grouping variable
